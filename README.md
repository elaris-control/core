# 🤖 elaris-home / core

The open-source engine of the **elaris** ecosystem. A powerful, AI-driven automation framework designed for home and industrial environments.

---

### 🌟 Overview
**elaris-core** is the central nervous system of our automation platform. It provides the essential logic, database schemas, and API integrations needed to build a smart, responsive, and secure environment.

### 🛠 Key Features
* **AI-First Design:** Deep integration with advanced LLMs via Claude Code & OpenRouter.
* **Modular Architecture:** Built to support various IO nodes and hardware configurations.
* **Scalable Engine:** Designed for both simple home tasks and complex industrial automation.
* **Open Core:** The fundamental engine is public and open for community inspection.

### 🏗 Tech Stack
* **Runtime:** Node.js / Python environment.
* **Interface:** Command Line Interface (CLI) optimized for developer efficiency.
* **Connectivity:** Support for ESP32 firmware and MQTT/Websocket protocols.
